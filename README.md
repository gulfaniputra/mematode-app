## About

A spartan meme generator web application.

The name 'mematode' derives from the portmanteau of 'meme' & 'nematode'.

## Description

The objective is to learn ES6 & React features, especially:

- array destructuring
- arrow function
- spread operator
- computed property key
- `&&` operator
- ternary operator
- props & state
- `useState`
- event handlers
- `<input>` component
- `useEffect`
